🦄 Infinity Quiz

Infinity Quiz is a quiz game that I made to learn more about OpenAIs API. It is randomizing question from category, and subcategory tuples, and provide answers with different personalities ad infinitum.

👁️ Proofreader

After spending lots of time proofreading Github README's, me and a collegue wanted to automate the process. Proofreader opens a pull request for every push and reviews the README, coming with suggestions and corrections. We used OpenAIs API with function calls to decide when a correction was due, and we integrated the corrections into Githubs GUI.

It does work, but the utility is arguably quite low in it's current state (look at this PR🤪). It is spewing suggestions for every paragraph (or are we just poor writers?). The corrections are not idempotent, so it just keeps on correcting already proofread paragraphs. The project is on hold for now, but could be fixed and extended to incorporate code (e.g, automatic code repairing with pretty pull requests).

🎶 Symbolic Music Generation

A project made for the course DD2424 Deep Learning in Data Science. We coded two models (LSTM and Transformer) from scratch in PyTorch to generate MIDI, and trained on a pop music dataset. We used MLFlow to monitor and export the model, experimenting with various features such as MLproject-files (e.g, Makefiles for ML-projects). With some boundaries on the data, we managed to produce something similar to music. Looking at the Github repo, it should be reproducible from any machine. At the final poster session, it was crowned second coolest project of our course iteration! 🤩
